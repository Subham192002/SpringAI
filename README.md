# Spring Boot ChatGPT Clone

A simple ChatGPT-like application built using **Spring Boot** and the **OpenAI API**, with a dark-themed chat UI.

This project demonstrates how to:
- Expose a REST API with Spring Boot  
- Call an AI model from Java  
- Build a lightweight browser-based chat interface  
- Keep API keys secure using environment variables
